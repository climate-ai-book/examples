Key Concepts and Mathematical Foundations

1. Seasonal Anomaly Decomposition

Forced + Internal Variability:

X(t) = X_forced(t) + X_internal(t) = f(B(t)) + ε(t)Where:

X_forced: predictable component from boundary conditions B
X_internal: unpredictable atmospheric variability
B: SST, soil moisture, snow cover

2. ML Seasonal Prediction

Conditional Probability:

p(X_{t+τ} | I_t, C_t) = f_ML(I_t, C_t; θ)Inputs:

I_t: initial conditions (atmospheric state)
C_t: slowly-evolving climate state, external forcings
θ: learned parameters

3. RNN for Climate Evolution

Temporal Dynamics:

h_t = tanh(W_h·h_{t-1} + W_x·x_t + b_h)
y_{t+τ} = W_y·h_t + b_yHidden state h_t encodes climate memory

4. Ensemble Seasonal Prediction

Adaptive Weighting:

X̂_ensemble = Σᵢ wᵢ(τ, conditions)·X̂ᵢWeights vary with lead time and climate regime

5. LSTM for Sub-Seasonal Forecasting

Complete LSTM Cell:

f_t = σ(W_f·[h_{t-1}, x_t] + b_f)  # Forget gate
i_t = σ(W_i·[h_{t-1}, x_t] + b_i)  # Input gate
C̃_t = tanh(W_C·[h_{t-1}, x_t] + b_C)
C_t = f_t ⊙ C_{t-1} + i_t ⊙ C̃_t  # Cell state
o_t = σ(W_o·[h_{t-1}, x_t] + b_o)  # Output gate
h_t = o_t ⊙ tanh(C_t)

6. Hybrid Prediction

Dynamical + ML Correction:

X̂_hybrid(t+τ) = X̂_dynamical(t+τ) + Δ_ML(X̂_dynamical, F_features, τ)

7. Attention Mechanisms

Predictor Importance:

αₜ = exp(eₜ) / Σₜ' exp(eₜ')
eₜ = v^T tanh(W·hₜ)Attention weights identify relevant time steps/locations

8. Probabilistic Precipitation

Category Classification:

P(Precip > threshold | X_predictors) = f_classifier(X_predictors)Classes: above-normal, near-normal, below-normal

9. Multi-Model Ensemble

GCM Integration:

P̂_MME = g_ML({P̂ᵢ^GCM}ᵢ, C_indices, O_obs)Combines multiple climate models with indices and observations

10. Climate Driver Integration

Regional Prediction:

P̂_regional(r,t+τ) = h_ML(ENSO(t), IOD(t), SAM(t), SST_patterns(t), H_historical(r))Large-scale drivers:

ENSO: El Niño-Southern Oscillation
IOD: Indian Ocean Dipole
SAM: Southern Annular Mode

11. Temperature Downscaling

GCM to Field Scale:

T_field(x,y,t+τ) = f_downscale(T_GCM(t+τ), E_elevation(x,y), L_landcover(x,y), T_local-obs)Growing Degree Days:
GDD_forecast = Σₜ max(0, T_daily(t) - T_base)

12. Drought Index Forecasting

SPI Prediction:

SPI(t+τ) = f_ML(P_historical(t), F_climate-forecast(t), V_vegetation(t), S_soil-moisture(t))Multi-category severity:
P_drought-categories(t+τ) = Softmax(f_neural(X_predictors(t)))

13. Monsoon Prediction

Basin-Scale Rainfall:

P_monsoon(basin, season) = g_ensemble(I_ENSO, I_IOD, S_SST, M_GCM, H_historical)

14. ML Downscaling

Coarse to Fine Scale:

Y_local(x_fine, y_fine, t) = f_downscale(X_GCM(x_coarse, y_coarse, t), T_terrain, L_land, O_obs-hist)CNN Spatial Upscaling:
Y_high-res = CNN_upscale(X_low-res, W_learned)

15. Bias Correction

Quantile Mapping + ML:

X_corrected = F_obs^{-1}(F_model(X_model)) + Δ_ML(conditions)

16. Hybrid Framework

Lead-Time Dependent Weighting:

X̂_hybrid = α(τ)·X̂_dynamical + (1-α(τ))·X̂_ML + β·Δ_correctionCollaborative Prediction:
X̂_{t+τ}^collaborative = f_ML(X̂_{t+τ}^dynamical, E_ensemble-spread, S_skill-metrics, C_conditions)

17. Adaptive Ensemble Weighting

Skill-Based Combination:

X̂_merged = Σₘ wₘ^ML(S_skill, C_conditions)·X̂ₘModel Selection:
S_m(r,s,τ) = g_ML(Skill_historical(m,r,s,τ), C_current, M_model-characteristics)

18. Streamflow Forecasting

Multi-Input Framework:

Q(t+τ) = f_ML({Q(t-k)}_k, W_weather(t), C_climate(t), S_snow(t), B_basin)Seasonal Water Supply:
V_seasonal = ∫_{t_spring}^{t_fall} Q(t) dt = g_ML(SWE_peak, C_seasonal-forecast, P_historical)

19. SHAP for Interpretability

Feature Attribution:

φⱼ(x) = Σ_{S⊆F\{j}} [|S|!(|F|-|S|-1)!]/|F|! · [f(S∪{j}) - f(S)]Quantifies predictor importance

20. Physics-Constrained Loss

Conservation Enforcement:

L_total = L_prediction + λ_physics·L_physicsEnsures physical consistency (water/energy balance)
