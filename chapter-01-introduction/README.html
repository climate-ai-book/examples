Key Concepts and Mathematical Foundations

1. Climate System Dynamics

Navier-Stokes Equations:

∂v/∂t + (v·∇)v = -(1/ρ)∇p + ν∇²v + f

Continuity Equation:

∂ρ/∂t + ∇·(ρv) = 0

Thermodynamic Equation:

∂T/∂t + v·∇T = (1/ρc_p)(Dp/Dt + Q)

2. Information Theory for Climate

Mutual Information:

I(X;Y) = Σ_x Σ_y p(x,y)log(p(x,y)/(p(x)p(y)))

Entropy:

H(X) = -Σ p(x)log p(x)

Conditional Entropy:

H(X|Y) = H(X,Y) - H(Y)

Cross-Entropy Loss:

L_CE = -Σ_i y_i log(ŷ_i)

3. Neural Network Architectures

Convolution Operation:

(f * g)[m,n] = Σ_i Σ_j f[i,j] · g[m-i,n-j]

Translation Equivariance:

Conv(T_τ(x)) = T_τ(Conv(x))

LSTM Cell State Updates:

f_t = σ(W_f·[h_{t-1}, x_t] + b_f)      # Forget gate
i_t = σ(W_i·[h_{t-1}, x_t] + b_i)      # Input gate
C̃_t = tanh(W_C·[h_{t-1}, x_t] + b_C)  # Candidate
C_t = f_t * C_{t-1} + i_t * C̃_t       # Cell state

Continuous-time LSTM analog:

dC/dt = f(t)·(-C) + i(t)·C̃(t)

4. Physics-Informed Neural Networks

PINN Loss Function:

L = L_data + λ_pde L_pde + λ_bc L_bc + λ_ic L_ic

Mass Conservation:

∂ρ/∂t + ∇·(ρv) = 0

Energy Conservation:

∂T/∂t + v·∇T = κ∇²T + S

Additional Physical Constraints:

Hydrostatic: ∂p/∂z = -ρg
Ideal Gas: p = ρR_d T
Geostrophic: fk×v_g = -(1/ρ)∇_h p

Heat Equation Residual:

R_heat = ∂u/∂t - α∇²u - f(x,t)

5. Multi-Scale Analysis

Wavelet Transform:

W(a,b) = (1/√a)∫ f(t)ψ*((t-b)/a) dt

Morlet Wavelet:

ψ(t) = π^{-1/4} e^{iω_0 t} e^{-t²/2}

Dilated Convolution:

y[n] = Σ_k x[n - dk]·w[k]

Power-Law Scaling:

E(k) ∝ k^{-β}
(β ≈ 5/3 for atmospheric turbulence)

Feature Pyramid Network:

P_l = F_l + Upsample(P_{l+1})

6. Transfer Learning

H-Divergence (Domain Discrepancy):

d_H(S,T) = 2 sup_{h∈H} |Pr_S[h(x)=1] - Pr_T[h(x)=1]|

Domain-Adversarial Training:

L_total = L_task(D_s) - λL_domain

7. Uncertainty Quantification

Bayes' Theorem:

p(w|D) = p(D|w)p(w)/p(D)

KL Divergence:

KL[q(w;θ)||p(w|D)] = ∫ q(w;θ)log(q(w;θ)/p(w|D)) dw

Evidence Lower Bound (ELBO):

ELBO = E_{q(w;θ)}[log p(D|w)] - KL[q(w;θ)||p(w)]

Ensemble Methods:

ȳ = (1/N)Σ_{i=1}^N y_i
σ² = (1/N)Σ_{i=1}^N (y_i - ȳ)²

Monte Carlo Dropout:

μ̂ = (1/T)Σ_{t=1}^T f(x; w_t)
σ̂² = (1/T)Σ_{t=1}^T f(x; w_t)² - μ̂²

8. Interpretability Methods

Gradient-Based Attribution:

A_i = (∂y/∂x_i)·x_i

Integrated Gradients:

IG_i(x) = (x_i - x'_i)∫_{α=0}^1 (∂f(x'+α(x-x'))/∂x_i) dα

Attention Weights:

α_i = exp(e_i)/Σ_j exp(e_j)

Self-Attention:

Attention(Q,K,V) = softmax(QK^T/√d_k)V

9. Performance Metrics

Regression Metrics:

MAE = (1/n)Σ|y_i - ŷ_i|
RMSE = √[(1/n)Σ(y_i - ŷ_i)²]
NSE = 1 - Σ(y_i - ŷ_i)²/Σ(y_i - ȳ)²

Classification Metrics:

F1 = 2·(Precision·Recall)/(Precision + Recall)
AUC = ∫_0^1 TPR(FPR^{-1}(x)) dx

10. Future Paradigms

Foundation Model Loss:

L_foundation = Σ_i λ_i L_i + L_physics + L_consistency

Hybrid Physics-ML System:

∂u/∂t = F_physics(u) + F_ML(u; θ)
